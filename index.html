About this Site

Here's some information for you, Heather! Click the questions to reveal the answers.

What is this site?

This site is a prototype site that will allow learners to see available L&D badges at my company, learn how to earn badges, and see which badges they have already earned. This site will become a page in my company's internal Sharepoint site. As you do not have access to our internal site, I made this page a standalone on Github so you can see it.

Who made this site and in what program?

This site was entirely hand-coded by myself (Rebeccca) from scratch including the HTML, CSS, and JavaScript/jQuery. I have a backgroud in web design from both my undergrad and my professional life. The program was in Notepad ++ because it's what I was most comfortable with back in my web design days.

How does it know the person viewing it earned specific badges?

At the moment, the functionality for the site to actually detect and track each user's progress in earning badges is not present. This requires a much higher level of coding including SQL, Python, and heavy use of APIs through both Skilljar, Microsoft Teams, and more. Conveniently, this is my husband's wheelhouse so he has verified that it is possible and I can produce scripts and a Tableau dashboard that shows that the data is able to be successfully retrieved from Skilljar as an example. I won't unless you require it because it might be more technical than you need to judge the project. However this is a very large effort and we are still in talks about timing and compensation. This project is a real, living thing that will continue after LDT 505 is over.

Why does it think I earned some badges and not others?

For now, I have awarded you some badges so you can see what it'll look like when we have all of the functionality described in the previous qustion completed. To prove this site can be updated per user, your badge awards aren't hard-coded into the site. Instead, the list is in a .txt file this site is pulling from. Eventually, it will pull from an SQL server (which is within our capabilities as well!)

Why is the information on this page about the class?

The information on this page will be replaced to describe the site to learners without discussing the Penn State course that started it. But I thought the functionality could be communicated here while still keeping the information helpful to the instructor reviewing it (Hello, Heather! I hope you like my site!)

Why this style for the site and badges?

This site and all graphics in it are using official company colors, images, and icons. The style also reflects the style of the company's website.

The links don't work!

Sorry, they will lead to our company's Skilljar portal, which does not allow guest users. I can send screenshots, if needed! Skilljar is mobile-friendly.

Action-Related Badges

Earn these badges by performing specific actions

Minto Master

Earned by delivering a presentation to a client using the Minto Pyramid Principle.

How to Earn

X

Deliver a presentation, either internally or to a client, using the Minto Pyramid Principle described in the Presentation Skills course. Ensure your AL submits a confirmation that you have completed this task to the AL channel in Microsoft Teams.

Opportunity Spotter

Earned by submitting a Data & Analytics opportunity.

How to Earn

X

Submit a Data & Analytics opportunity (a lesson to do this can be seen in the Intro to Data & Analytics course in Skilljar.

Skilljar Learner

Earned by completing 5 courses in Skilljar.

How to Earn

X

Complete a total of any 5 courses currently available in Skilljar. It can be any course that is available.

Expert Skilljar Learner

Earned by completing 10 courses in Skilljar.

How to Earn

X

Complete a total of any 10 courses currently available in Skilljar. It can be any course that is available.

Course Reviewer

Earned by submitting feedback on at least 5 courses in Skilljar.

How to Earn

X

Complete a feedback form for any 5 courses in Skilljar.

External Learner

Earned by using part or all of your $1300 external training stipend.

How to Earn

X

Submit a request to use your external training stipend and have the request approved.
